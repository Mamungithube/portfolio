E-Learning Platform

Full-stack online course management system with enrollment & progress tracking

View on GitHub

Project Overview

This E-Learning Platform is a personal project I built to deepen my Django skills while solving a real-world problem — creating a complete online education system from scratch. It covers the full lifecycle of online learning, from course creation by instructors to student enrollment, lesson delivery, and progress tracking.

The project focuses heavily on REST API design, role-based access control (instructors vs students), and database modeling for complex relationships like many-to-many enrollments, nested course content, and per-student progress tracking.

Key Features

Course & Lesson Management

Instructors can create courses with multiple sections and lessons. Each lesson supports text content, video URLs, and attachments. Full CRUD with drag-and-drop ordering via API.

Student Enrollment System

Students can browse the course catalog and enroll in courses. Enrollment supports free and paid courses, with payment integration for premium content access.

Progress Tracking

Each lesson completion is tracked per student. The system calculates overall course completion percentage and marks courses as complete when all lessons are done.

Reviews & Ratings

Enrolled students can leave reviews and star ratings for courses they've enrolled in. Average ratings are calculated dynamically and exposed on course listing endpoints.

Role-Based Access Control

Three roles — Admin, Instructor, and Student — each with different permissions. Instructors only manage their own courses; students only access enrolled content; admins control everything.

Course Discovery & Filtering

Searchable course catalog with filtering by category, difficulty level, instructor, and price range. Full-text search on course titles and descriptions.
